# kuracchi's WezTerm Config

[WezTerm](https://wezfurlong.org/wezterm/) の設定ファイルです。

## ファイル構成

```
~/.config/wezterm/
├── wezterm.lua   # メイン設定
└── keybinds.lua  # キーバインド設定
```

---

## `wezterm.lua` — メイン設定

### 基本設定

| 設定 | 値 | 説明 |
|---|---|---|
| `font_size` | `12.0` | フォントサイズ |
| `use_ime` | `true` | IME（日本語入力）を有効化 |
| `window_background_opacity` | `0.75` | ウィンドウ背景の透過率 |
| `macos_window_background_blur` | `20` | macOS のウィンドウ背景ブラー強度 |
| `automatically_reload_config` | `true` | 設定ファイル変更時に自動リロード |

### タブバー

- タイトルバーを維持しつつリサイズ可能に設定（`TITLE | RESIZE`）
- タブが1つのときはタブバーを非表示
- タブバー・タイトルバーを透過し、背景と一体化
- 新規タブボタン・タブ閉じるボタンを非表示
- タブのデザインをカスタマイズ
  - アクティブタブ: 金色（`#ae8b2d`）
  - 非アクティブタブ: グレー（`#5c6d74`）
  - Nerd Fonts の三角形アイコンでタブ左右を装飾

### キーバインド

デフォルトキーバインドを無効化し、`keybinds.lua` で定義したキーバインドを使用。

---

## `keybinds.lua` — キーバインド設定

### 入力

| キー | 動作 |
|---|---|
| `Shift + Enter` / `Alt + Enter` | 改行（`\n`）を送信 |

### コピー / 貼り付け

| キー | 動作 |
|---|---|
| `Cmd + C` | クリップボードにコピー |
| `Cmd + V` | クリップボードから貼り付け |

### タブ操作

| キー | 動作 |
|---|---|
| `Cmd + T` | 新しいタブを開く |
| `Cmd + W` | 現在のタブを閉じる（確認あり）|
| `Cmd + Q` | アプリを終了 |
| `Cmd + Alt + ←` / `→` | 前/次のタブへ移動 |
| `Ctrl + Tab` | 次のタブへ移動 |
| `Shift + Ctrl + Tab` | 前のタブへ移動 |
| `Cmd + 1〜8` | 番号指定でタブへ移動 |
| `Cmd + 9` | 最後のタブへ移動 |

### ペイン操作

| キー | 動作 |
|---|---|
| `Cmd + H` | ペインを左に分割 |
| `Cmd + J` | ペインを下に分割 |
| `Cmd + K` | ペインを上に分割 |
| `Cmd + L` | ペインを右に分割 |
| `Cmd + Shift + X` | 現在のペインを閉じる（確認あり）|
| `Cmd + Shift + P` | ペイン選択モード |
| `Cmd + Z` | 現在のペインをズーム（他ペインを非表示）|

### その他

| キー | 動作 |
|---|---|
| `Cmd + Shift + C` | コピーモードを起動 |
| `Ctrl + +` / `-` / `0` | フォントサイズ 拡大 / 縮小 / リセット |
| `Cmd + P` | コマンドパレットを開く |
| `Cmd + Shift + R` | 設定を手動リロード |
| `Cmd + Shift + W` | ワークスペース選択ランチャーを表示 |

### コピーモード（`Cmd + [` で起動）

vi ライクなキーバインドで、スクロールバック内のテキストを選択・コピーできます。

| キー | 動作 |
|---|---|
| `h` / `j` / `k` / `l` | 左 / 下 / 上 / 右 に移動 |
| `w` / `b` / `e` | 単語単位で前進 / 後退 / 末尾へ |
| `f` / `F` | 指定文字へジャンプ（前方 / 後方）|
| `t` / `T` | 指定文字の直前へジャンプ（前方 / 後方）|
| `0` / `^` / `$` | 行頭 / 行の最初の非空白 / 行末へ移動 |
| `g` / `G` | スクロールバックの最上部 / 最下部へ移動 |
| `H` / `M` / `L` | ビューポートの上端 / 中央 / 下端へ移動 |
| `Ctrl + B` / `F` | ページアップ / ダウン |
| `Ctrl + U` / `D` | 半ページ上 / 下にスクロール |
| `v` | 文字単位の範囲選択 |
| `V` | 行単位の範囲選択 |
| `Ctrl + V` | ブロック単位の範囲選択 |
| `y` | 選択範囲をクリップボードにコピー |
| `Enter` | コピーして終了 |
| `Escape` / `q` / `Ctrl + C` | コピーモードを終了 |
